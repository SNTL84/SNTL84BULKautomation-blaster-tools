# 🤝 Contributing to SNTL84 BULK Automation Blaster Tools

Thank you for your interest! This project welcomes contributions from automation enthusiasts, marketers, and developers.

---

## 📋 How to Contribute

### 1. Fork & Clone
```bash
git clone https://github.com/SNTL84/SNTL84BULKautomation-blaster-tools.git
cd SNTL84BULKautomation-blaster-tools
```

### 2. Create a Feature Branch
```bash
git checkout -b feature/your-feature-name
```

### 3. Make Your Changes
- Open the HTML file in any browser to test
- No build tools or npm required — pure HTML/JS

### 4. Commit with a Clear Message
```bash
git commit -m "feat: describe your change"
```

### 5. Push & Open a Pull Request
```bash
git push origin feature/your-feature-name
```
